# Meme Gallery

Advanced Web Application Development assignment.

## Development from git

1. ```composer update --no-scripts```

## How to run

1. ```composer install```

2. ```php artisan make:database meme_gallery_db```

3. ```php artisan migrate --seed```

4. ```php artisan serve```
